# Contributing Guidelines

Welcome! We are excited to have you contribute to the CV Parsing and Scoring project. Here are some guidelines to help you get started:

## How to Contribute
1. **Fork the repository**: Create a personal copy of the repository on your GitHub account.
2. **Clone the forked repository**: Use `git clone` to get a local copy of your fork.
3. **Create a new branch**: Use a descriptive name for your branch (e.g., `feature/your-feature-name`).
4. **Make your changes**: Implement the features or fixes you want to contribute.
5. **Commit your changes**: Write clear, concise commit messages.
6. **Push your changes**: Push your branch to your forked repository on GitHub.
7. **Create a pull request**: Go to the original repository and create a pull request comparing your branch with the main branch.

## Code of Conduct
- Be respectful and constructive.
- Review others' contributions and provide feedback.
- Be open to feedback on your contributions.

Thank you for contributing! We're looking forward to working with you!
